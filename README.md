# Focus Day

A lightweight, single-file planning app I built for myself to manage tasks around
ADHD/distractibility. Instead of an endless backlog, it forces a small, prioritized
"focus day": a few tasks, ranked by energy/priority, with a calm interface designed
to reduce overwhelm.

## Why
I get distracted easily, and traditional to-do apps made it worse — long lists,
notifications, and guilt. Focus Day keeps only what matters today in front of me,
color-codes tasks by priority (now / soon / later), and saves the day's plan to JSON.

## Files
- `focus-day.html` — the current single-file app (open in a browser)
- `adhd-todo.html` — earlier prototype
- `organize/` — a componentized version (`index.html`, `app.js`, `styles.css`)
